# Портал планирования свадьбы

Комprehensive веб-портал для управления всеми аспектами планирования свадьбы.

## 📋 Описание

Портал планирования свадьбы — это полнофункциональное приложение, разработанное для упрощения процесса организации свадьбы. Оно помогает парам подготовить их особенный день, предоставляя инструменты для управления гостями, отслеживания бюджета, координации с подрядчиками и многое другое — всё на одной централизованной платформе.

## ✨ Основные возможности

- **Управление гостями** — Создание, редактирование и управление списками гостей с отслеживанием RSVP и учётом пищевых предпочтений
- **Планировщик бюджета** — Установка бюджетов, распределение средств по категориям и мониторинг расходов в реальном времени
- **Координация с подрядчиками** — Хранение контрактов, отслеживание платежей и управление коммуникацией
- **Планировщик рассадки** — Визуальный инструмент с drag-and-drop для создания схемы рассадки за столами
- **Хронология и задачи** — Чеклисты, важные вехи и напоминания о сроках
- **Уведомления** — Email и push-уведомления о важных событиях

## 🚀 Начало работы

### Предварительные требования

- Node.js >= 18.x
- npm или yarn
- PostgreSQL >= 14
- Docker и Docker Compose (опционально)

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd wedding-planning-portal
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Настройте переменные окружения:
   ```bash
   cp .env.example .env
   # Отредактируйте .env, указав учетные данные базы данных и настройки
   ```

4. Запустите базу данных:
   ```bash
   docker-compose up -d
   ```

5. Выполните миграции базы данных:
   ```bash
   npm run migrate
   ```

6. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

7. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере
